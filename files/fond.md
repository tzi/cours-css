# Fond

## Image de fond

```css
.image {
	background-image: url('photo.jpg');
}
```

La fonction `url()` permet d'indiquer l'url d'une image pour l'utiliser comme fond d'un élément.

## Taille d'une image

```css
.image {
	background-image: url('photo.jpg');
	background-size: cover;
}
```
